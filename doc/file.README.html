Overview

The Create module realizes resources defined in Hiera data by using create_resources.

Module Description

The Create module realizes resources defined in Hiera data by using create_resources.

Setup

What Create affects

Resources will be created on the system.

Setup requirements

  • Puppet-3.0.0 or later

Beginning with Create

Include '::create::(type)' in your manifests and create hiera data with a key 'create_(type)' to create resources of 'type'.

Usage

include '::create::users'

create_users: joe: managehome: true shell: /bin/bash
